Projects

Simon's game using Java

A Simon's Game in Java using Swing for the GUI. A 2x2 grid of colored buttons (red, green, blue, yellow) using `JButton`. Randomly generate a sequence of button flashes. Capture user input via button clicks (`ActionListener`) and validate it against the sequence. Feedback provides through color changing and progressively increase difficulty by extending the sequence. The score is displayed at the middle and manage game states (start, win, or lose).
